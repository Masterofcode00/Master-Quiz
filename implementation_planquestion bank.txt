# Goal Description
Add the ability to bulk upload a question bank via Excel or CSV on the Admin Dashboard. 

## User Review Required
> [!WARNING]
> While you asked for **PDF or Excel**, I strongly recommend sticking to **Excel (.xlsx) or CSV** formats for now. PDF files are unstructured documents meant for reading, not for data extraction. Extracting Questions, 4 Options, and the Correct Answer reliably from a PDF is very error-prone. Excel and CSV files are structured into rows and columns, guaranteeing 100% accurate data imports. 
> 
> My plan below implements Excel/CSV upload. It will expect the Excel file to have columns like [Question](file:///c:/Users/Varun%20Kumar/OneDrive/Desktop/master%20quiz/Master-Quiz/New/frontend/js/dashboard.js#159-197), `Option A`, `Option B`, `Option C`, `Option D`, and `Correct Answer` (A, B, C, or D).
> Please confirm if this approach works for you!
